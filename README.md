# sql-data-analytics-project

This repository contains SQL scripts for advanced data analytics, including time-based trends, cumulative analysis, performance metrics, part-to-whole analysis, and customer/product segmentation.

Each script represents a specific business use case and demonstrates practical SQL techniques using CTEs and window functions.

## 📊 Project Structure

The project is organized into multiple SQL scripts, each focusing on a specific analytical pattern:

- **Time-Based Trends**  
  Analyze changes in key metrics over time.

- **Cumulative Analysis**  
  Calculate running totals and track growth.

- **Performance Analysis**  
  Evaluate customer and product performance using KPIs.

- **Part-to-Whole Analysis**  
  Measure contribution of categories or products to overall results.

- **Segmentation**  
  Group customers and products based on behavior and value.

- **Customer & Product Reports**  
  Generate structured reports with key metrics.

  ## 🛠️ Skills & Techniques

- Advanced SQL
- CTEs (Common Table Expressions)
- Window Functions (e.g., LAG, SUM OVER)
- Aggregations & KPI calculations
- Data Segmentation
- Time-Series Analysis
